## ☕ The Project
**Artisan** is a modern, fast, and elegantly animated website built to deliver a premium digital experience for a neighborhood cafe. It replaces generic templates with a custom-engineered interface, complete with a dark-mode-inspired warm beige aesthetic, fluid entrance animations, and a seamless client-side cart system.

## ✨ Features
- **Premium Aesthetics & Motion:** Fully custom UI with robust, responsive layouts and stagger-fade micro-animations powered by `framer-motion`.
- **Dynamic Store Hours:** A robust, logic-driven real-time hours indicator that intelligently displays "Open Now", "Closes Soon", or exactly exactly when the shop opens next.
- **Client-Side Cart:** Persisted Shopping Cart state via React Context that updates in real-time across routes.
- **Smart Menu Filtering:** Fluid, animated menu navigation allowing users to filter items by category (Coffee, Food, Drinks, Seasonal) on the fly.
- **Fully Responsive:** Painstakingly designed to look pixel-perfect on both ultra-wide monitors and small mobile screens.

## 🛠 Tech Stack
This project leverages the modern React ecosystem for aggressive performance and type-safety:
- **Framework:** [Next.js](https://nextjs.org/) 15 (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) v4
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Language:** TypeScript

## 🚀 Running Locally

Want to inspect the code and run the cafe yourself? 

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Pratyaksh1507/cafe-site.git
   cd cafe-site
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the application:**
   Open [http://localhost:3000](http://localhost:3000) in your browser.
